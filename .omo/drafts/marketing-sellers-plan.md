# Draft: Marketing and User Acquisition Plan for Mercado Libre Sellers

## 1. Goal & Success Criteria
- **Objective**: Attract active Mercado Libre sellers to use the MeliOps application, converting them from free users to paid subscribers (Pro/Infinity).
- **Success Criteria**: 
  - Increase organic web traffic.
  - Convert active sellers to register (using Google OAuth).
  - Convert active users to Pro (4.990 CLP) and Infinity (12.990 CLP) subscriptions.

## 2. Core Value Propositions
1. **Huge Cost Savings**: Group 6 shipping labels on 1 A4 sheet (2x3 grid) -> Save up to 83% adhesive label paper. No expensive thermal printers needed.
2. **Pack Error Reduction (Picking Integrated)**: Light gray box under label shows exact product description (quantity, color, size). Pack correct item every time without separate lists.
3. **Massive Time Savings**: Bulk process multiple PDFs. Ready in seconds. No more tedious manual recortar/cropping.
4. **Zero Hassle**: Immediate web-based use, free tier available (5/day, 20/mo), no complex installs or ERP integrations.

## 3. SEO & Content Strategies (Based on Existing Research)
- **High-Volume Keywords**: 
  - `imprimir etiquetas mercado libre` (Vol: 2400, KD: 18)
  - `separar etiquetas mercado libre` (Vol: 140, KD: 5)
- **Proposed Deliverables**:
  - Educational TOFU guide on "Cómo separar e imprimir etiquetas de Mercado Libre para ahorrar 80% papel".
  - Interactive savings calculator on the landing page (Input: monthly packages -> Output: adhesive paper saved, CLP saved, hours saved).
  - Landing page CRO improvements (showcase picking box benefit).

## 4. Open Questions for the Seller Acquisition Plan
- **Primary Market / Region**: Is focus 100% on Chile (CLP) or MLA (Argentina), MLB (Brazil), MLM (Mexico) too?
- **Channels**: Organic SEO/Content vs Guerrilla marketing (Facebook groups, WhatsApp, Mercado Libre forums) vs Paid Ads.
